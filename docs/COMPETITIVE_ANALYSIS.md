## 二、竞品全景（2026 年现状）

### 调研对象分类

| 类别 | 工具 | 定位 |
|------|------|------|
| **全栈 ASM 框架** | reNgine、ARL、Nemo、NetProbe | 自托管，端到端资产发现平台 |
| **资产测绘 SaaS** | FOFA、Shodan、Censys、Hunter | 商业，全网索引，无主动扫描 |
| **单点工具** | nuclei、httpx、OneForAll、Amass | 专注一个环节，需串联 |
| **桌面端** | Goby | 资产+漏洞一体，无 Web 服务 |

### 竞品现状更新（关键变化）

- **ARL（灯塔）**：官方仓库 [TophantTechnology/ARL] 已下架停更，社区备份版 [Aabyss-Team/ARL] 和改进版 [adysec/ARL] 维持可用，内置 7000+ 指纹仍是强项
- **reNgine**：2.0 版活跃维护，是**国际开源领域定位最接近 NetProbe 的全栈框架**（子域名→端口→漏洞→端点收集），但无中文情报源、无风险评分
- **Nemo（Go 版）**：[hanc00l/nemo_go] 国产自托管，集成 nmap/masscan/nuclei，但功能较基础
- **Goby**：仍为桌面端（无 Web 服务/API/持久化），资产+漏洞一体是强项

## 四、NetProbe 的独占优势（全行业唯一）

以下能力**没有任何单一竞品具备**，是 NetProbe 的核心护城河：

| 独占能力 | 说明 | 最近竞品 |
|---------|------|---------|
| 🔴 **六维变更检测 Diff** | 跨扫描自动 diff 端口/Web/技术栈/敏感路径/JS/banner + 时间线趋势 | 无（Shodan 有历史快照但非自动 diff） |
| 🔴 **智能风险评分闭环** | 6 维加权 0-100 分，风险驱动排序而非扁平列表 | Goby 有简单评分，无多维度 |
| 🔴 **JS/SPA 分析 + 敏感路径 + 指纹版本提取** 组合 | 从前端 bundle 提 API/密钥 + 566 条敏感路径 + 组件版本号 | reNgine 仅端点收集 |
| 🔴 **中英文情报融合** | FOFA+Hunter(中文) + crt.sh+Shodan+Censys(国际) 一站式 | 国际工具无中文源，国内工具无国际源 |
| 🔴 **一站式管道 + 持久化 + 告警** 闭环 | 子域名→端口→Web→指纹→漏洞→报告 全流程，结果持久化可追溯 | reNgine/ARL 接近但无 diff/评分 |

## 五、NetProbe 的短板与改进路线

| 短板 | 现状 | 影响竞品差距 | 改进方向（v3.0） |
|------|------|------------|----------------|
| ⚠️ **通知渠道单一** | 仅 Webhook | reNgine 有 Discord/Slack/Telegram 多通道 | 补邮件/钉钉/企微/飞书/TG |
| ❌ **无 Docker 部署** | 手动起进程 | reNgine/ARL/Nemo 均支持 | docker-compose 一键启动 |
| ❌ **无多用户** | 无认证 | reNgine/ARL 有 RBAC | 用户认证+权限 |
| ⚠️ **指纹规则数** | 961 条 | ARL 7000+ / FOFA 数万 | 兼容 Wappalyzer 格式导入社区库 |
| ❌ **无 API 资产发现** | 仅 JS 提取 | 痛点 #8 | REST/GraphQL 端点主动发现 |
| ❌ **无插件系统** | 单体 | — | 模块化扩展 |

### 优先级建议（按"缩小竞品差距"排序）

1. **Docker 部署** — 工作量小，立刻让 NetProbe 达到 reNgine/ARL 的部署体验门槛
2. **多渠道通知** — 工作量小，告警维度从 ⚠️ 变 ✅
3. **指纹库扩充（Wappalyzer 兼容）** — 拉平与 ARL 的指纹数量差距
4. **API 资产发现** — 补全攻击面，强化独占优势
5. **多用户 / 插件系统** — 架构级，长期项

## 六、定位结论

**NetProbe 是开源领域能力维度最全的一体化资产探测平台**（16 维中拿下 14 个 ✅），在变更检测、风险评分、JS 分析、中英文情报融合四个维度具备**全行业独占优势**。

当前主要差距集中在**工程化**（Docker/多用户）和**指纹规模**，而非核心能力。补齐 Docker + 多渠道通知 + 指纹库扩充三项后，NetProbe 将在能力矩阵上对所有开源竞品形成全面领先。
